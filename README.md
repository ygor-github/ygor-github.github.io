# Portafolio de Ygor Hernández

## Descripción

Este es el código fuente del portafolio personal de Ygor Hernández, un profesional en transición hacia el campo de la Ciencia de Datos. El sitio web muestra su perfil, formación, certificaciones y formas de contacto.

## Características

- **Multilenguaje:** Soporte para Español, Inglés y Portugués.
- **Diseño Responsivo:** Adaptable a diferentes tamaños de pantalla (escritorio, tablet y móvil).
- **Interfaz Moderna:** Construido con Tailwind CSS para un diseño limpio y moderno.
- **Contenido Dinámico:** El contenido de texto se carga desde un archivo JSON, facilitando la adición de nuevos idiomas y la gestión del contenido.

## Tecnologías Utilizadas

- **HTML5:** Para la estructura del sitio web.
- **CSS3 (Tailwind CSS):** Para el diseño y estilo.
- **JavaScript:** Para la funcionalidad de cambio de idioma y la interactividad.
- **Font Awesome:** Para los iconos.

## Cómo Usar

Para ver el sitio web, simplemente abre el archivo `index.html` en tu navegador.

Si deseas realizar cambios:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Ygor-Hernandez/Portfolio.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd Portfolio
   ```
3. Abre `index.html` en tu navegador o edita los archivos (`index.html`, `js/main.js`, `data/translations.json`) según sea necesario.
